Personal Blog

I decided to undertake the project of building my own blog post from scratch where I not only note but also share my experiences and knowledge in everyday things. I humbly chose to consistently and periodically share my daily experiences so that anyone else can freely learn the very same things if they are interested.

I blog about programming, physical computing, capentry, musical instruments and fitness and wellness. Only these do I engage myself in 100% of the time. I cannot hold any conversation outside these listed items.

I have no other social media presence other than the necessary work sites such as GitHub and local programming communities near me.

Features:

  • Ability to post comments
  • Realtime Date and Time
  • Payment
  • Live Language Translation
  • Consultation Scheduling
  • Database Management

Tools Used:

  • Calendly for Scheduling
  • Microsoft Language Translator
  • Flask Framework
  • Twitter Bootstrap for Cross-browser Responsiveness
  • Python3 for Programming
  • Heroku for Deployment
  • SQLite for Local Development
  • Stripe For Payment
  • Moment JS for Realtime Date and Time
  • Gravatar for User Comments
  • Google reCaptcha for Extra Form Security
  • JQuery for Mobile Responsiveness
  • Ngrok for Localhost Testing

Tinker Chat App

Somasoma Blog

This is a blogging platform that an admin can use to share ideas or knowledge that he or she has. Featured items include the admin's personal interests and courses they find most interesting. Users can check out general information about available courses but will need to log in before they can view specific course information. It has been built using Python and Flask.
